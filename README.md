# 🚀 Rules for Shadowrocket

这里托管的是我自用的 Shadowrocket 配置文件及分流列表。

---

### 📂 文件结构说明

| 文件名 | 作用说明 | 更新逻辑 |
| :--- | :--- | :--- |
| **`cycle1337_sr_rule.conf`** | **主配置文件**。包含所有策略组（✨Gemini、♻️ 自动优选等）和全局设置。 | 手动推送生效 |
| **`Personal_DIRECT.list`** | **直连白名单**。确保 B 站上传、500px 访问及局域网设备（如 Tank 300）跑满物理带宽。 | **即时同步** |

| **`gemini_rules.list`** | **Gemini 专项**。锁定 Google AI 域名，确保其始终通过“美国住宅”节点访问。 | **即时同步** |
| **`ip_Check.list`** | **IP 检测**。确保访问检测类网站时显示代理节点 IP。 | 24小时更新 |

### 🛠️ 核心优化点

* **智能 DNS**：整合了腾讯及阿里 DoH 解析，兼顾国内访问速度与隐私保护。
* **AI 避障**：通过严格的正向/负向正则过滤，物理隔离香港节点对 Gemini 的干扰。
* **硬核户外适配**：针对局域网（Lan.list）进行优化，确保连接车载系统或内网设备时不产生握手延迟。
* **自动化流**：配置了 GitHub Actions，每次修改规则都会在文件头部自动刷新北京时间戳。

---

### 📥 导入与使用

1.  复制本仓库中 `.conf` 文件的 **Raw 链接**。
2.  打开 Shadowrocket，在“配置”页面选择“从 URL 下载”。
3.  **重要提示**：导入后请在“配置”选项卡中长按该文件，选择“文本编辑”，确保你的 HTTPS 解密证书（MITM）已正确安装并信任。
